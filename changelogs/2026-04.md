# 变更日志 - 2026年4月

## 2026-04-11 (下午)

### 优化

#### 系统性能调优
- 创建 `/etc/sysctl.d/99-server-optimize.conf` 内核优化配置
- 降低 Swap 使用倾向（swappiness: 60 → 10）
- 优化 TCP 连接效率（Fast Open、连接复用、缓冲区）
- 提高系统稳定性和网络性能

#### Docker 容器资源限制
- SanctionList backend: 内存限制 256MB，CPU 0.5核
- SanctionList frontend: 内存限制 64MB，CPU 0.25核
- Homepage: 内存限制 64MB，CPU 0.25核
- 防止单个容器耗尽系统资源

#### 日志管理
- Docker 容器日志轮转配置（最大 10MB，保留 7 天）
- systemd-journald 日志限制（最大 100MB，保留 7 天）

#### 监控增强
- 健康检查脚本 v2.1
- 新增 OOM（内存溢出）检测
- 新增 TCP 连接状态统计
- 新增容器内存使用率监控
